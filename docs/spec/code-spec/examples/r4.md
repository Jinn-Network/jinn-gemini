# r4 — Enforce automated secret guardrails

## ✅ Compliant automation

```ts
import { gitGuard } from '../git/guard.js';

export async function autoCommit(message: string) {
  await gitGuard.ensureSafeStagedTree();
  await git.commit({ message });
  await git.push();
}
```

- Every commit path invokes the shared guard before touching git history.
- The guard blocks `.operate/`, `.operate-test/`, and `*private_key*` files automatically.

## ❌ Violation

```ts
export async function autoCommit(message: string) {
  await git.commit({ message });
  await git.push();
}
```

- Skips the canonical guard, so staged secrets slip through unchecked.
- Violates r4 because automation pushes without enforcing the denylist.
