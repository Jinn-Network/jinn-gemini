# Gemini CLI Configuration

This is the main Gemini CLI configuration file for the Jinn project. This file provides context and instructions for general development work using the Gemini CLI.

## Project Overview

This is the Jinn project - a system for automated job processing using AI agents. The project includes:

- **Frontend Explorer**: A Next.js web application for exploring data
- **Gemini Agent System**: Automated job processing with telemetry
- **Metacog MCP Server**: Custom tools for database operations
- **Worker System**: Background job processing

## Development Guidelines

When working with this project:

1. **Use appropriate tools**: The project has custom MCP servers and tools available
2. **Follow project structure**: Understand the different components and their purposes
3. **Check documentation**: Review the `docs/` directory for detailed information
4. **Respect agent system**: The `gemini-agent/` directory contains specialized agent configurations

## Available Tools

The project includes several MCP servers and tools:
- Metacog MCP Server: Database operations and job management
- Context7: Documentation lookup
- Sequential Thinking: Problem-solving assistance
- Various web and file system tools

## Important Notes

- The `gemini-agent/GEMINI.md` file is specifically for automated agent execution
- This file (`GEMINI.md`) is for general CLI development work
- Check the `docs/` directory for detailed project documentation
