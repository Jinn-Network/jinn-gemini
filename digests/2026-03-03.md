- Convention-over-config for jinn-node: JSON config file with defaults, auto-generated from env vars on first run, full backwards compatibility

## Architecture decisions

- Staked mech list now built reactively from `ServiceStaked` events rather than static `CreateMech` lookback — handles new services without restarts
- IPFS metadata cached in Postgres rather than in-memory — survives reindexes, which was the primary cause of 3+ hour backfills
- Private IPFS network deprioritized in favor of shipping operator tooling this week
- DevOps infrastructure: explicit decision to not invest in formal release structure or CI/CD overhaul right now — "strategically muddle along" until tooling catches up
